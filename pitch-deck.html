What's Inside

Price & Chart
  • Live BTC / USD from Kraken
  • 24h % change with arrow
  • 24h OHLCV chart + VWAP line
  • Hi · Lo · Cap · Sat/$
News & Lead Story
  • Drop-cap lead with inline image
  • Bitcoin Magazine · CoinDesk · BTC.com
  • BREAKING · MARKETS · MINING
  • Scrollable headlines + excerpt
Chain Vitals
  • Block height + avg block time
  • Hashrate · difficulty · retarget ETA
  • Mempool size & tx count
  • Fee estimates: fast / eco
Weather & Forecast
  • Current conditions + 8-hour forecast
  • Animated Meteocons SVG icons
  • Auto dark mode at local sunset
  • °F / °C · location configurable
Mining Fleet
  • Solo odds: 1-in-X per day
  • Per-miner hashrate, power, temp
  • Efficiency (J/TH) · uptime
  • Fleet settings via localStorage
Editorial
  • On This Day: Bitcoin milestones
  • Proof of Read: live price + chain TL;DR
  • Diamond (♦) bullet summary panel
  • Updates from live data each refresh

Plus: live feed-health status dots · scrolling chain vitals ticker · dark/light mode toggle

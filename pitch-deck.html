The Technology

1 file.
0 installs.
∞ Bitcoin.
Single-file React / Babel build — JSX transpiles in-browser, zero bundler at runtime.
5 live data sources: Kraken · Mempool.space · Open-Meteo · RSS2JSON · BitAxe (local).
12 custom hooks · 25 components · responsive: 1920×1080 desktop, fluid mobile.
Optional Python proxy — solves BitAxe CORS, serves dashboard + miner API same-origin. Multi-arch Docker image on GHCR.
Zero cloud. Zero subscription. ~376 KB built. Open source. MIT licensed.
